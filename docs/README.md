# Documentation

Comprehensive documentation for the Nadi CakePHP SDK, a monitoring package that integrates
the [Nadi](https://nadi.pro) platform into CakePHP applications.

## Documentation Structure

### [01. Getting Started](01-getting-started/README.md)

Installation, setup, and basic usage of the Nadi CakePHP SDK.

### [02. Architecture](02-architecture/README.md)

System design, data flow, and architectural patterns used in the SDK.

### [03. Configuration](03-configuration/README.md)

Configuration reference for drivers, sampling, security, and environment variables.

### [04. Monitoring](04-monitoring/README.md)

Detailed guides for exception tracking, HTTP monitoring, and slow query detection.

### [05. Advanced](05-advanced/README.md)

OpenTelemetry integration, custom extensions, and advanced usage patterns.
